# Android Codelabs - Jetpack Compose

Este repositorio contiene las soluciones a los primeros codelabs del curso "Android Basics with Compose" de Google.

## Proyectos

### 1. GreetingCard
* **Descripción:** Una app estática que muestra un mensaje de texto.
* **Conceptos:** Funciones `@Composable`, `Text` y uso básico de `Modifier`.

### 2. HappyBirthdayCard
* **Descripción:** Muestra texto superpuesto sobre una imagen de fondo.
* **Conceptos:** Layout `Box` para apilar elementos, `Image`, `painterResource` y alineación con `Modifier.align`.

### 3. DiceRoller
* **Descripción:** Una app interactiva con un botón que "lanza" un dado y actualiza la imagen.
* **Conceptos:** Gestión de estado con `mutableStateOf` y `remember`, `Button` y el evento `onClick`, y lógica condicional (`when`) para actualizar la UI (Recomposición).

## Stack Tecnológico

* **Lenguaje:** Kotlin
* **UI:** Jetpack Compose
* **IDE:** Android Studio
